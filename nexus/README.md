

## Nexus 설치시

### Step 1. env 파일 설정

- email 과 your-ip-address 를 a record 로 등록한 domain 정보를 기입

###  Step 2. Nexus 설치

```console
  $ chmod +x start-nexus.sh
  $ ./start-nexus.sh
```

### Step 3. Potainer 설정

- Portainer 는 5분 내에 admin 계정을 생성해야 사용 가능하다. 

```
  Portainer - http://your-ip-address:9000
```
